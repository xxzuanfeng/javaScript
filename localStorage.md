# Learn localStorage

```js
element.classList.toggle("class-to-toggle");					//.toggle方法切换属性，不存在则添加属性，存在则移除
dialogElement.showModal();									//modal模式在未解除前阻止与外部元素所有交互
dialogElement.close();										//modal模式关闭
numbers.findIndex((num) => num > 3);						//findIndex方法返回满足条件的索引
str.toLowerCase();											//toLowerCase方法把字符串全部小写
Date.now()													//返回从1970开始的毫秒数
const arr = [1, 2, 3];
arr.unshift(0);												//.unshift()方法把元素添加到开头[0,1,2,3]
fruits.splice(1, 2);									//.splice(arg1,arg2),第一个参数为开始位置，第二个参数为删除的数量，并返回删除元素。
```
